# DNS Settings

## Scope
- Configure DNS API providers and CNAME domains.
- Validate domains and strip http/https prefixes when inputting CNAMEs.

## UI
- DNS provider panel with credential fields.
- CNAME domain list (add/edit/delete).

## API
- CRUD for DNS providers and CNAME domains.
- Return provider types and required credential fields.

## Node/Sync
- DNS configuration affects automated record creation for line/region routing.
