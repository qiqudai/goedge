# Node Management (API)

## Scope
- Node CRUD, monitoring settings, and status reporting.
- Agent heartbeat, config pull, and task execution.

## UI
- Node list with status, bandwidth, traffic, and actions.
- Monitor logs dialog and realtime views.

## API
- Admin endpoints: list/create/update/delete nodes.
- Agent endpoints: heartbeat, config fetch, task reporting.

## Node/Sync
- Sync node config changes through API; update Nginx/OpenResty config.
- Use reload/hot-reload after config changes.
