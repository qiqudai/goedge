# Line Management

## Scope
- Define line categories and mapping to DNS line IDs per provider.
- Support default, domestic, overseas, search engine, and custom lines.

## UI
- Display line definitions and their provider-specific mappings.
- Provide selection in resolution configuration.

## API
- Read-only list for frontend selection and DNS API mapping.

## Node/Sync
- Line mapping is used when creating DNS records for resolution.
