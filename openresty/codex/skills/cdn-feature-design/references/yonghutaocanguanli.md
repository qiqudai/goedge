# User Package Management

## Scope
- Assign base packages to users with start/end times.
- Track usage limits and renew/upgrade flows.

## UI
- List assigned packages with usage stats.
- Modals for assign/renew/upgrade.

## API
- Assign package to user; set start/end time.
- Return package details for usage view.

## Node/Sync
- Package changes that affect runtime limits must sync to nodes.
