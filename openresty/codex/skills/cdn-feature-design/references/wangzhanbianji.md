# Site Editing

## Scope
- Manage domain list, origins, protocols, caching, headers, and security for sites.

## UI
- Add/edit site dialog with domain and origin inputs.
- Advanced settings: cache rules, backsource, headers, HTTPS, CC/ACL, gzip, websocket.

## API
- CRUD for sites; batch updates.

## Node/Sync
- Sync site config to nodes and generate per-domain server/upstream config.
